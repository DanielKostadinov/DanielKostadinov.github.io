Bootcamp 2020: RandomLightning

My first programming project, concluding the 2 week Programming Bootcamp at City (2020). I created a program what creates a colourful line that moves randomly with the help of noise methods at the position of the mouse when the "a" button is pressed. Also when the "lightning" is halfway across the screen you can press the mouse button to "charge" the individual sparks and make them move differently. All of that was achieved with the help of a implemented class - spark, nested "for" loops and "if" methods. Finaly by pressing the "ENTER" button you can stop the program from spawning more lightning.
